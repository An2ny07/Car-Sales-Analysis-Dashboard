# Car Sales Analysis Dashboard
A data analytics project designed to analyze car dealership performance using a 23,000-row dataset. This project includes a fully interactive Power BI dashboard that visualizes key sales metrics, customer insights, company trends, and regional performance. Excel used for processing and Power BI for building KPIs, DAX measures, and dynamic visuals.

## Project Overview
This project analyzes car dealership sales performance using Excel for initial data preparation and Power BI for dynamic dashboard visualization.
The dashboard helps track critical KPIs, uncover trends, and support data-driven decision-making across various car models, regions, and body styles.

## Objective
To design and develop an interactive Car Sales Dashboard that provides real-time insights into sales performance, pricing trends, customer preferences, and regional sales distribution.

## Tools Used
- Microsoft Excel – dataset review, cleaning, preprocessing
- Power BI Desktop – data modeling, DAX measures, visualization
- DAX – KPI calculations (YTD, MTD, YOY metrics)
- Power Query – cleaning & transformation
- Excel Dataset: 23,000+ rows of car sales transactions
